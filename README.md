# Анализ больших данных — Лабораторная работа №1

**Тема:** Нормализация данных в аналитическую модель «Снежинка» (Snowflake Schema).

В данном репозитории реализован автоматизированный ETL-процесс на Python, который загружает сырые данные из CSV-файлов в PostgreSQL и нормализует их до Третьей нормальной формы (3NF), распределяя по таблицам фактов и измерений.

---

## Инструкция по запуску

### 1. Запуск базы данных
Убедитесь, что у вас установлен Docker. Запустите контейнер с PostgreSQL:
```bash
docker-compose up -d
```

### 2. Установка зависимостей
```bash
pip install psycopg2-binary
```
### 3. Запуск ETL скрипта
```bash
python main.py
```
